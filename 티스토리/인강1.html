컴퓨터 프로그래밍 이해

프로그래밍 정의

프로그램 : 하드위어를 동작시켜 특정 작업을 수행하는 소프트웨어
프로그래밍 : 인간이 컴퓨터에 명령하는 작업지시서를 만드는 작업
프로그래밍 언어 : 작업 지시에 사용되는 언어

고급언어(high-level)

인간 중심의 언어(c, java, python)를 해당 하드웨어에 맞게 번역하는 프로그램을 이용

1. 유지보수가 비교적 쉬움
2. CPU 동작을 구체적으로 알아야 할 필요없음
3. 하드웨어마다 별도의 프로그래밍 불필요

저급언어(low-level)

기계 중심의 언어(기계어, 어셈블리어(middle-level))를 CPU가 처리하는 명령어와 1:1로 대응

1. CPU마다 사용하는 명령어가 다름
2. CPU 내부 동작을 이해해야 프로그래밍이 가능
3. 하드위어에따른 별도 프로그래밍이 요구됨 - 불편함

프로그래밍 절차

프로그래밍은 통역사와 비슷하며, 인간의 언어를 컴퓨터가 이해할 수 있게 번역하는 기능을 수행함

컴파일러

컴퓨터가 이해하는 언어로 번역하는 프로그램, 기계마다 별도의 프로그램이 불필요

C언어 프로그래밍 절차

소스파일 .C (확장자)→ (컴파일러에 의해)오브젝트파일 .OBJ(확장자) →(Linker에 의해) 실행파일 .exe(윈도우 환경)
[ 코드 작성→컴파일→링크→실행 ]

C언어

1. 1972년 AT&T의 벨연구소의 데니스리치와 켐톤슨에 의해 개발
2. B언어를 기반으로 하여 개발
3. 고급언어인 C언어로 작성된 최초의 운영체제가 UNIX
4. 최초의 언어C: ANSI C

C언어의 특징

1. 간결하고 효율적인 언어
2. 우수한 이식성
3. 다소 배우기가 어려움

통합개발환경(IDE)

1. 프로그램 개발에 필요한 S/W
2. 텍스트 편집기, C컴파일러

통합개발환경(IDE) 소개

1. 프로그램 개발 절차를 하나의 프로그램에서 수행되도록 제공하는 환경
2. 소스코드 작성 컴파일, 링크, 실행, 디버깅 등을 하나의 프로그램에서 실행

통합개발환경 종류

Visual Studio, Dev C++, Code:Blocks, IDE one
