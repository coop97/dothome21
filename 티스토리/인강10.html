문자열

문자열

null로 끝나는 문자들의 집합
1. 문자열은 큰 따옴표("")로 표현함
2. 문자열의 끝에는 널 문자('\0')를 함께 저장함

문자열 기초

문자열 상수

값이 변경되지 않는 문자열
-"A"나 "Hello World"등
-'문자열 리터럴'이라고도 함

문자열 변수

프로그램 수행 중에 변경될 수 있는 문자열

선언

문자 배열의 크기는 저장할 문자 수 +1 크기로 지정
-> 널 문자가 들어가야하기 때문

초기화

" "문자를 대입연산자에 이용
* 배열 크기 -1개의 문자를 초기화 가능

문자열 표준 입력

scanf()

1. 입력된 문자열을 str 배열에 저장
2. 공백 또는 enter 전까지 입력된 문자열을 저장
3. 함수 형식: int scanf("%s", str);

gets()

1. 공백을 포함한 문자열 입력
2. char* gets(char* str);
3. enter를 '₩0'으로 변환하여 저장

문자열 표준 출력

printf()

형식 문자열 "%s"를 이용
문자 배열명을 printf함수의 첫 번째 인자로 가능

puts()

1. int puts(const char* str);
2. str 배열의 문자열을 표준 출력에 출력
3. 배열의 '₩0'을 '₩n'로 변환하여 출력
