조건문

if

조건을 판단하여 참인 경우 문장 수행
*조건에 따른 수행 문장이 두 문장 이상인 경우는 { }로 시작과 끝을 알린다.(한 문장이면 생략 가능)

if ~ else

조건 외의 문장을 수행

f ~ else if ~ else

조건을 여러 개 넣을 경우

switch문

조건 판단문으로서 조건에 따라서 여러가지 작업을 수행
else if 와 비슷한 기능을 가짐

break

블럭({ })을 탈출하는 명령어

default

else와 비슷한 기능을 가짐

입력문

scanf()

1.scanf()를 이용한 입력
2.표준입력으로부터 다양한 자료를 지정한 변수에 저장
3.형식지정자(%d %c %f %lf)사용
4.공백,enter전까지를 입력
5.형식:scanf("형식지정자", &변수명)

정수 입력

형식지정자: %d

실수 입력

형식지정자: %f, %if(double형 데이터를 입력할 때 씀)

문자 입력

형식지정자: %c, 단일문자
*fflush (stdir); : enter가 문자로 들어가는 것을 방지하기 위해 쓰는 함수(buffer를 비우기 위해 사용함)
