반복문

같거나 유사한 코드를 반복적으로 수행할 수 있도록 하는 제어문
for문과 while문을 사용

for문

기본 형식

for(초기식; 조건식; 증감식)
반복할 문장;

*for문을 구성하는 초기식, 조건식, 증감식, 반복할 문장은 모두 생략할 수 있음
*무한 루프문 : for(;;)

while문

기본 형식

while(조건식)
반복할 문장;

for와 while

for를 while로 변경할 때는 while문 앞에 초기식을 쓰고 while 블록 안쪽의 맨 끝에 증감식을 씀

do while

한 번은 반드시 문장을 수행해야하고 그 다음에 조건을 적용시킬 때 사용함

기본 형식

기본 형식: do{
반복할 문장;
}while(조건식);

분기문

break : 블록을 탈출하는 명령어
continue : 반복문의 시작으로 분기
