프로그램 기본 구조

main()함수

함수의 정의

입력에 대해 연산을 수행학 결과를 출력, 프로그램에서 처리할 내용을 모아두는 묶음
입력x → 함수(f) → 출력y
y=f(x)

1. C언어는 하나 이상의 함수로 이루어짐
2. 실행 가능한 하나의 프로그램에는 반드시 하나의 main()함수가 존재함
3. C언어는 main()함수를 수행하는 언어

기본구조

프로그램 기본 틀

헤더파일(라이브러리 사용을 위한 준비) / 프로그램 시작점 / 프로그램 종료
소스파일→(전처리기)→(컴파일러)→오브젝트 파일→(링커)→실행파일

코드 작성 준비

1. 소문자로 작성해야 함
2. 식별자는 대소문자를 구분
3. 함수의 시작과 끝은 {}로 구분
4. 하나의 문장 끝은 ;

출력문장 구현

주석문

프로그램 내에서 코드 등을 설명하기 위한 용도로 사용, 프로그래머에게만 보이고 컴퓨터에서는 처리되지 않는 문장
여러 줄 주석 : /* ········ */
한 줄 주석: //

출력문

1. printf(" ");
2.표준출력장치(화면)에 ""에 지정한 글자를 나타내는 함수
3.헤더파일 stdio.h에 선언(printf(" ");을 사용하기 위해 stdio.h를 반드시 써야함

이스케이프 문자

/n : 커서를 다음 줄로 이동
＼＼: ₩ 문자를 화면에 출력
＼" : " 문자를 화면에 출력
＼' : ' 문자를 화면에 출력
＼b: 커서를 한 칸 이전으로 옮김
₩n : 다음줄로 이동
