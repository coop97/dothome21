연산자

숫자

피연산자 +-기호들: 연산자

산술연산자

산술연산에 사용되는 연산자 사칙연산(+, -, *, /), %(나머지)

관계연산자

크기 비교연산에 사용되는 연산자, 비교 결과 참(true)이면 1, 거짓(false)이면 0
>, <, >=, <=, !=(다른가), ==

논리연산자

AND, OR, NOT 연산에 사용되는 연산자
&&, ||(엔터 위에 있는 기호, 탭하면 \이 나옴), !

증감연산자

값을 1만큼 증가 또는 감소시키는 연산자
증감 연산자는 반드시 변수에만 사용
연산자 위치에 따라 연산 순서가 다름

비트연산자

비트단위의 연산에 사용되는 연산자
&(AND), |(OR), ~(NOT), ^(XOR exclusive OR)

이동연산자

이동 후 빈자리는 0으로 채워짐
비트 왼쪽 이동: 왼쪽 피연산자 *2의 n승
비트 오른쪽 이동: 왼쪽 피연산자 /2의 n승

대입연산자

왼쪽 변수에 오른쪽 값을 저장
왼쪽은 변수만 사용가능
복합대입 연사자를 사용할 수 있음

조건연산자

삼항 연산자
수식C(조건) ? 수식T(조건이 참인 경우의 값) : 수식F(조건이 거짓인 경우의 값)

형변환연산자

1.묵시적 형변환: 컴파일러에 의해 자동으로 수행하는 형변화(올림변환, 내림변환)
2.명시적 형변환: 지정한 데이터 타입으로 임시적으로 수행하는 형변환
