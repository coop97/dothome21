변수와 데이터형

변수

프로그램 수행 시 데이터를 보관할 수 있는 공간

1.메모리에 저장공간 확보
2.방법에 따라 크기와 용도가 다르게 생성
3.유일한 이름 부여
4.메모리에 대한 심볼릭 표현

변수선언

변수는 사용 전 반드시 먼저 선언해야 함
필요 시 선언과 동시에 초기화도 가능함

변수선언문 기본형식

데이터타입 변수명;
데이터타입 변수명1, 변수명2 ····· ;
변수명은 식별자 생성 규칙을 적용함

식별자

변수를 구분하기 위해서 사용되는 이름

1. 식별자는 반드시 영문자, 숫자, 밑줄 기호(_)만을 사용해야 함
2. 식별자의 첫 글자는 반드시 영문자 또는 밑줄 기호(_)로 시작해야함(숫자는 안됨)
3. 식별자에는 밑줄 기호(_)를 제외한 다른 기호를 사용할 수 없음
4. 식별자는 대소문자를 구분해서 만들어야 함 (name, Name, NAME 전부 다른 글자)
5. C 언어의 예약어는 식별자로 사용할 수 없음 예약서 EX)if, print, switch 등

변수 활용

초기화는 =(대입연산자)를 이용함
변수명=값;

상수

변하지 않는 자료값, 보관되는 값에 따라 의미가 변하는 것이 변수

리터럴 상수

정수형, 실수형, 문자형

심볼릭 상수

기호를 가지고 의미를 부여 (const 상수, 매크로 상수, 열거형 상수)

매크로 상수

#define문으로 정의되는 상수

형식

#drfine 매크로이름 상수
ex)#drfine TOTAL 300

데이터타입

C프로그램에서 사용되는 모든 변수나 상수 값은 정해진 데이터형을 가짐
데이터형에 의해서 주어진 값의 이진 표현이 결정됨
기본 데이터형(문자형, 정수형, 실수형)
파생 데이터형(배열, 포인터)
사용저 정의형(구조체, 공용체, 열거체)

문자형

저장하는 데이터가 문자인 경우
char :부호있는 문자형(1byte)
unsigned char: 부호없는 문자형(1byte)

* c언어에서 단일문자는 단일 인용부호를 씀

정수형

c언어는 정수형으로 short(2byte), ≤int(4byte), ≤long(4byte)을 제공함
long long(8byte)
signed : ", 생략가능
unsigned : "

정수의 표현

1.부호가 있는 정수형은 최상위 비트를 부호 비트로 사용함
2.부호가 있는 정수형은 음수를 표현하는데 2의 보수를 사용함
3.부호가 없는 정수형은 최상위 비트를 값을 저장하는 용도로 사용함

실수형

실수의 표현 방식

1.고정소수점 방식(정수, 소수로 구분)
2.부정소수점 방식(기수, 지수로 구분)

실수의 정밀도

1.float형
2.double형(8byte)
3.long double

형식지정자

형식지정자를 이용한 printf() 출력
%u : 부호 없는 십진수
%e, %E : 지수형식 출력
%f, %lf : 실수값 출력
%s : 문자열 출력
