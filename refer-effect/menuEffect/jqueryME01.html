margin

margin 속성은 박스 요소의 바깥 여백을 일괄적으로 설정합니다.

margin-top

margin-top 속성은 박스 요소의 위쪽에 바깥 여백을 설정합니다.

margin-right

margin-right 속성은 박스 요소의 오른쪽에 바깥 여백을 설정합니다.

margin-bottom

margin-bottom 속성은 박스 요소의 아래쪽에 바깥 여백을 설정합니다.

margin-left

margin-left 속성은 박스 요소의 왼쪽에 바깥 여백을 설정합니다.

const tabBtn = $(".tab-btn ul li");                                                            
const tabCont = $(".tab-cont > div"); 

tabBtn.click(function(){
    let target = $(this);
    tabBtn.removeClass("active");
    target.addClass("active");

    let index = target.index();
    tabCont.css("display","none");
    tabCont.eq(index).css("display","block");
});
